# GreenTechHub - Reclamation Module

This repository contains the Reclamation module for the Web Development project.
The project follows MVC architecture, uses PDO, CSRF protection, and server-side validation.

To run:
- Clone into XAMPP/htdocs
- Start Apache & MySQL
- Import the database
- Visit http://localhost/greentechhub/public/
